# Mental-health-workforce-dashboard
Power BI dashboard analyzing mental health and workforce trends in healthcare.
An interactive Power BI dashboard designed to explore mental health trends in the healthcare workforce, including stress levels, burnout, turnover, and employee absences.

## Project Overview
This dashboard helps analyze workforce wellbeing across departments and employee types. It gives a clear view of how stress and burnout relate to absences and turnover in the healthcare environment.

## Data Source
- Source: Healthcare Workforce Mental Health Dataset
- Format: CSV
- Contains employee-level workforce data such as department, employee type, stress level, burnout risk, absences, and turnover indicators.

## Tech Stack
- Power BI Desktop
- Power Query
- DAX
- CSV Dataset

## Features / Highlights
- KPI cards for key workforce metrics
- Department-wise stress analysis
- Burnout distribution visualization
- Turnover trend and rate analysis
- Absence comparison by employee type
- Interactive slicer-based filtering

## Business Problem
Healthcare organizations face rising employee stress, burnout, and turnover. This dashboard helps identify which departments and employee groups are most affected so leaders can take data-driven action.

## Goal of the Dashboard
- KPI cards show the main dashboard metrics at a glance.
- **Stress chart** highlights departments with higher average stress levels.
- **Burnout donut chart** shows the distribution of burnout categories.
- **Turnover chart** compares turnover count and turnover rate.
- **Absence chart** compares average absences by employee type.

## Business Impact & Insights
- Reveals departments with high stress and burnout risk
- Helps identify employee groups with higher turnover tendency
- Supports HR and leadership in improving workforce wellbeing
- Provides a visual foundation for retention and intervention strategies
